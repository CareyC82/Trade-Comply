# TraceWize Demo Scenarios

Use these five scenarios for customer demos, short videos, and regression checks. They are intentionally route-specific and high-tech focused.

1. AI GPU accelerator chip: United States to China, import opportunity.
2. Solar panel photovoltaic: China to United States, destination import compliance.
3. Energy storage battery system: China to European Union, destination import compliance.
4. Industrial robot arm: Germany to United States, destination import compliance.
5. 800G optical transceiver module: China to India, destination import opportunity.

Source data: `data/demo-scenarios.json`.
