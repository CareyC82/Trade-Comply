# TraceWize 60-Second Product Demo Video Script

## Video Goal

Create a concise product demo that shows how TraceWize helps trade, compliance, and operations teams run an import/export compliance pre-check from product selection to actionable report.

Target length: 60 seconds  
Tone: professional, clear, modern, compliance-focused  
Format: screen recording with voiceover and light captions  

## Core Message

TraceWize helps teams quickly pre-screen global import/export compliance risks across product categories, trade routes, compliance focus, source monitoring, HS classification, rule matches, checklists, and reports.

## Voiceover Only

Meet TraceWize, an AI-assisted compliance pre-check platform for global import and export workflows.

Trade compliance teams need to check customs rules, export controls, certifications, tariffs, cases, and documentation across scattered official sources.

TraceWize monitors official sources daily, filters weak matches, and turns updates into structured compliance signals.

Start by choosing an industry, trade direction, and counterparty market. For example, screen a solar panel import into China from the United States.

TraceWize matches the product against a structured rule library and highlights relevant compliance signals, source references, and trade-flow context.

It converts regulatory signals into practical next steps, including certification checks, customs documents, and shipment preparation tasks.

The HS Code assistant provides AI-assisted pre-classification, cross-border code context, and a reminder to verify final classification with a licensed broker.

Then generate a pre-screening report for internal review, compliance handoff, or shipment preparation.

TraceWize helps teams move faster, reduce blind spots, and stay aligned with changing trade rules.

## Editing Notes

- Keep cuts fast in the first 10 seconds.
- Use zoom highlights around: trade direction toggle, country dropdown, quick select card, matched rule count, checklist, and report button.
- Keep captions short. Avoid long legal copy on screen.
- Add a small disclaimer in the report/HS Code segment: "Pre-screening only. Final classification and legal decisions require professional review."
- Use calm background music and a steady voiceover pace.
